# MHP Fintech Dashboard - Installation Guide

## 📱 Install as Mobile App

Your MHP Fintech Dashboard is now a **Progressive Web App (PWA)** that can be installed on any device!

---

## For Android Users 📱

### Method 1: Chrome Browser
1. Open the dashboard URL in **Chrome**
2. You'll see a popup: **"Install MHP Fintech"**
3. Tap **"Install"**
4. The app will be added to your home screen
5. Open from home screen like any app!

### Method 2: Manual Install
1. Open dashboard in Chrome
2. Tap the **3-dot menu** (top right)
3. Select **"Add to Home screen"** or **"Install app"**
4. Tap **"Add"** or **"Install"**
5. App icon appears on home screen

---

## For iPhone/iPad Users 🍎

### Using Safari
1. Open dashboard URL in **Safari** (must use Safari, not Chrome)
2. Tap the **Share button** (box with arrow pointing up)
3. Scroll down and tap **"Add to Home Screen"**
4. Edit name if desired (default: "MHP Fintech")
5. Tap **"Add"**
6. App icon appears on home screen

**Note:** iOS doesn't show install prompts automatically - you must use the Share menu.

---

## For Desktop/Laptop 💻

### Chrome, Edge, or Brave
1. Open dashboard in browser
2. Look for **install icon** (➕ or computer icon) in address bar
3. Click it and select **"Install"**
4. App opens in its own window
5. Access from:
   - Windows: Start Menu or Desktop
   - Mac: Applications folder
   - Linux: Applications menu

### Alternative
- Click **3-dot menu** → **"Install MHP Fintech"**

## Uninstall Instructions

### Android
1. Long-press app icon
2. Select "Uninstall" or drag to "Remove"

### iOS
1. Long-press app icon
2. Tap "Remove App" → "Delete App"

### Desktop
1. Click 3-dot menu in app window
2. Select "Uninstall MHP Fintech"

## Troubleshooting

**"Install" option not showing?**
- Make sure you're using a supported browser (Chrome, Safari, Edge)
- Try refreshing the page
- Check if already installed

**App not working offline?**
- First launch requires internet
- Offline features work after first successful load

**Need help?**
- Contact support with your device and browser info
- Check browser console for errors (F12)
